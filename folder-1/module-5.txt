Cut-Test Processors Commands:
>> The cut command is used to to extract specific parts of text from each line of a file or command output.
# It works on the Characters 


cut --version : This command will give you version

EX: data.txt 
{  
    Sachin
    Rohit
    Virat
}

>> cut -c 1-3 data.txt
Sac
Roh
Vir 

>> cut -c1 3,5,7 fileName.txt  : This is used to get the respective 
numbers from the file.

 >> cut -c1-3 filename : This is used to get the character from starting to specified range

 >> cut -c1-3, 6-8 filename: This is used to get_first 3 characters and from 6-8 characters also.

 

 AWK : It is also one of the text processors command
 Awk is a utility/ language  designed for data extraction. Most of the time it is used to extract fields from a file or from an output.

> awk --version
> awk '{print $1}' fileName : This command is used to get the first column/fields data from an file.
> ls -l  | awk '{print $1, $3}' > This is used to get the list of 1and 3rd fields of the ls-l output.
> ls -l | awk '{print $NF}' : This is used to get the last fileds/column of an ls -l output.
> cat file | awk '{$2="Sachin"; print $0}'  = replace words fields words

# command to get the an line which is having length of specific bytes.
ex: awk  'length{$0} > 15' fileName


# To get the Number of fields from  each row in an file: ls -l | awk '{print NF}'


GREP/ EGREP : It is also one of the Text Processors Commands
> The grep command which stands for "global regular expressions print",
processess text line by line and prints any lines which match a specified pattern.

Note : This is used to search only single word from an file. 

> grep --version  : to show which version we are using
> grep --help : to get more information about this command.

syntax: grep < keyword_to_search > file_name.

> grep -c keyword filename : This is used to search for an keyword and get the count of that word.


> grep -i keyword fileName : This is used to search for an keyword with case sensitive.

> grep -vi keyword fileName: This is used to get the every word from the file expect the provided word from the given file.

Egrep : This is used to search two words from an file.

 > egrep -i "keyWord1|keyWord2"  fileName
