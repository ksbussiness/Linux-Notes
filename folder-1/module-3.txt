
Important Things to remember in Linux :
------------------------------------------- >

.> Linux has super-user account called root.
    * Root is the most powerful account that can create, modify, delete accounts and make changes to system configuration files.
    * Even can delete teh Os also.
    * Linux is case-sensitive system. Which means ABC is not same has abc.
    * Avoid using spaces when creating files and directories.
    * Linux kernel is not an operating system.It is a small software   within Linux operating system that takes commands from the users and pass them to system hardware or peripherals.
    * Linux is CLI,not GUI.
    * Linux is very flexible as compared to other OS.


.> Access to Linux System.
    * There are 2 types of access.
     Console 
     Remote.
    # The console is  a direct access to an operating system when, it is  connected through VGA, HDMI, DVI.
    # The 2nd type of access is remote where you connect to your      operating system remotely over the network.

    Note : RDP : Remote Desktop Connection, connecting an os via IP address.

...> What is the Root ?

* There are 3 types of roots on Linux System.
  1) Root account: root is an account or a username on linux machine and it is the most powerful account which has access to all commands and files 
  2) Root as /: the very first directory in linux is also, referred as root directory.
  3) Root home directory : the root user account also has directory located in /root which is called root home directory.
